Executive Summary

1.1 核心判斷

AI 資料中心的瓶頸正在從「算力晶片」擴散到「資料搬運」。 GPU / XPU 數量增加後，cluster 內部需要更高頻寬、更低延遲、更低功耗的資料傳輸。傳統銅線與前面板可插拔光模組仍是主流，但在 800G、1.6T、3.2T 世代會逐步面臨功耗、訊號完整性、密度與散熱壓力。

短期主軸：800G / 1.6T 光模組放量。 AI cluster 仍大量依賴可插拔光模組，800G 是 2024–2027 年主力，1.6T 從 2025–2026 年導入、2027 年加速。

中期主軸：LPO / NPO 降低功耗與縮短電訊號距離。 LPO 減少模組端 DSP，NPO 將光學引擎靠近 ASIC，但仍保留一定模組化與維修彈性。

長期主軸：CPO / Optical I/O 把光互連推進封裝與系統核心。 CPO 將 optical engine 與 switch ASIC 緊密整合；Optical I/O 進一步將光互連推向 XPU package、rack-level、multi-rack scale-up interconnect。

2.2 數據結論

  • 近期主線：800G / 1.6T 可插拔光模組。 需求已反映在 AI cluster 建置與 hyperscaler 網路升級。
  • 中期成長：LPO / NPO / CPO。 目的不是追求新名詞，而是降低功耗、縮短電訊號距離、提高頻寬密度。
  • 長期方向：Optical I/O。 光互連將從交換器延伸到 XPU package、rack-level 與 multi-rack scale-up network。
  • 供應鏈瓶頸：雷射光源、光耦合、封裝測試、散熱與維修架構。

06

結論

AI 光通訊升級不是單一零件題材，而是資料中心架構升級。 800G / 1.6T 光模組是短期主線；LPO / NPO 是降低功耗與過渡到 CPO 的中期方案；CPO / Optical I/O 是長期把光互連推進封裝與系統核心的方向。

市場成長由 AI cluster 拉動，且數據已明確上修。 LightCounting、Yole、TrendForce、Omdia、Cignal AI 等機構的資料共同指向：AI 光互連、矽光子、CPO / NPO、高速光模組都會在 2025–2030 年進入結構性成長期。

台灣供應鏈的核心價值是整合。 台積電負責 SiPh / CPO 平台；聯鈞切入雷射封裝與光通訊封測；波若威、上詮、光聖切入高密度光纖連接與 FAU；系統廠承接 AI server、switch、rack integration。台灣不是只做單一零件，而是橫跨晶圓、封裝、光通訊、連接與系統整合的完整鏈條。
